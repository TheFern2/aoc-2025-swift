# Inputs Directory

This directory contains your Advent of Code puzzle inputs.

## Adding Input Files

1. Download your puzzle input from https://adventofcode.com/2025/day/X/input
2. Save it as `dayXX.txt` (e.g., `day01.txt`, `day02.txt`, etc.)
3. The input files are gitignored to comply with AoC's terms of service

## File Naming Convention

- Day 1: `day01.txt`
- Day 2: `day02.txt`
- ...
- Day 25: `day25.txt`

Use zero-padded day numbers (01-25).
